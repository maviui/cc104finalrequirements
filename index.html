10 Core Algorithms

1) Binary Search

Efficiently finds the position of a target value within a sorted array by repeatedly dividing the search interval in half.

  • Use cases: searching in sorted lists, databases, dictionaries.
  • Time: O(log n), Space: O(1)

2) Bubble Sort

Repeatedly swaps adjacent elements if they are in the wrong order. Simple but inefficient for large datasets.

  • Use cases: educational demos, nearly sorted arrays (optimized version).
  • Time: O(n^2), Space: O(1)

3) Merge Sort

Divide-and-conquer sorting that splits the list, sorts halves, and merges them. Stable and predictable complexity.

  • Use cases: large datasets, external sorting.
  • Time: O(n log n), Space: O(n)

4) Quick Sort

Divide-and-conquer sorting that partitions around a pivot. Excellent average performance, in-place.

  • Use cases: general-purpose sorting, in-memory sorts.
  • Time: Avg O(n log n), Worst O(n^2), Space: O(log n)

5) Dijkstra's Shortest Path

Finds shortest paths from a source to all vertices in a graph with non-negative edge weights.

  • Use cases: GPS routing, network routing, game AI.
  • Time: O((V+E) log V) with heap, Space: O(V)

6) Breadth-First Search (BFS)

Traverses graph level by level from the source; finds shortest path in unweighted graphs.

  • Use cases: social networks, shortest path unweighted, crawlers.
  • Time: O(V+E), Space: O(V)

7) Depth-First Search (DFS)

Explores as far as possible along each branch before backtracking. Useful for cycle detection and pathfinding.

  • Use cases: topological sort, connected components, puzzles.
  • Time: O(V+E), Space: O(V)

8) Kadane's Algorithm

Finds the maximum sum contiguous subarray in O(n) by dynamic tracking of current and best sums.

  • Use cases: stock analysis, signal processing.
  • Time: O(n), Space: O(1)

9) KMP String Matching

Searches for a pattern in text using a prefix table to skip re-checks, achieving linear time.

  • Use cases: text editors, DNA sequencing, search engines.
  • Time: O(n+m), Space: O(m)

10) Topological Sort

Linear ordering of a DAG’s vertices such that every directed edge u→v, u comes before v.

  • Use cases: task scheduling, build systems, course prerequisites.
  • Time: O(V+E), Space: O(V)

Pseudocode Library

Step-by-step pseudocode for each algorithm to reinforce understanding.

1) Binary Search

function binary_search(A, target):
  low ← 0
  high ← length(A) - 1
  while low ≤ high:
    mid ← floor((low + high) / 2)
    if A[mid] = target:
      return mid
    else if A[mid] < target:
      low ← mid + 1
    else:
      high ← mid - 1
  return -1

2) Bubble Sort

procedure bubble_sort(A):
  n ← length(A)
  repeat
    swapped ← false
    for i ← 1 to n-1:
      if A[i-1] > A[i]:
        swap A[i-1], A[i]
        swapped ← true
  until swapped = false

3) Merge Sort

procedure merge_sort(A):
  if length(A) ≤ 1: return A
  mid ← floor(length(A)/2)
  L ← merge_sort(A[0..mid-1])
  R ← merge_sort(A[mid..end])
  return merge(L, R)

procedure merge(L, R):
  result ← empty
  i ← 0; j ← 0
  while i < length(L) and j < length(R):
    if L[i] ≤ R[j]: append result, L[i]; i ← i+1
    else: append result, R[j]; j ← j+1
  append result, L[i..end]; append result, R[j..end]
  return result

4) Quick Sort

procedure quick_sort(A, lo, hi):
  if lo >= hi: return
  p ← partition(A, lo, hi)
  quick_sort(A, lo, p-1)
  quick_sort(A, p+1, hi)

procedure partition(A, lo, hi):
  pivot ← A[hi]
  i ← lo
  for j ← lo to hi-1:
    if A[j] ≤ pivot:
      swap A[i], A[j]
      i ← i+1
  swap A[i], A[hi]
  return i

5) Dijkstra's Algorithm

procedure dijkstra(G, source):
  dist[v] ← ∞ for all v
  dist[source] ← 0
  PQ ← min-priority-queue by dist
  insert (0, source) into PQ
  while PQ not empty:
    (d, u) ← extract-min(PQ)
    if d > dist[u]: continue
    for each (u, v, w) in neighbors(u):
      if dist[u] + w < dist[v]:
        dist[v] ← dist[u] + w
        insert (dist[v], v) into PQ
  return dist

6) BFS

procedure BFS(G, s):
  for each v in V: visited[v] ← false
  Q ← empty queue
  visited[s] ← true; enqueue(Q, s)
  while Q not empty:
    u ← dequeue(Q)
    for each v in neighbors(u):
      if not visited[v]:
        visited[v] ← true
        enqueue(Q, v)

7) DFS

procedure DFS(G, s):
  visited[v] ← false for all v
  call dfs_visit(s)

procedure dfs_visit(u):
  visited[u] ← true
  for each v in neighbors(u):
    if not visited[v]:
      dfs_visit(v)

8) Kadane's Algorithm

function kadane(A):
  best ← -∞
  current ← 0
  for x in A:
    current ← max(x, current + x)
    best ← max(best, current)
  return best

9) KMP String Matching

procedure compute_lps(P):
  lps[0] ← 0; len ← 0; i ← 1
  while i < length(P):
    if P[i] = P[len]: len ← len+1; lps[i] ← len; i ← i+1
    else if len ≠ 0: len ← lps[len-1]
    else: lps[i] ← 0; i ← i+1

procedure kmp_search(T, P):
  compute_lps(P)
  i ← 0; j ← 0
  while i < length(T):
    if T[i] = P[j]: i ← i+1; j ← j+1
      if j = length(P): report match at i-j; j ← lps[j-1]
    else if j ≠ 0: j ← lps[j-1]
    else: i ← i+1

10) Topological Sort (Kahn)

procedure topo_sort(G):
  indeg[v] ← 0 for all v
  for each edge (u, v): indeg[v] ← indeg[v] + 1
  Q ← all v with indeg[v] = 0
  order ← empty list
  while Q not empty:
    u ← dequeue(Q)
    append order, u
    for each (u, v) in edges:
      indeg[v] ← indeg[v] - 1
      if indeg[v] = 0: enqueue(Q, v)
  if length(order) ≠ |V|: report cycle
  return order

Array (List) Operations

Demonstrates basic array operations: insert, delete, search, and traversal.

# Array (Python list) operations
class Array:
    def __init__(self):
        self.data = []

    def insert(self, index, value):
        if index < 0 or index > len(self.data):
            raise IndexError("Index out of bounds")
        self.data.insert(index, value)

    def append(self, value):
        self.data.append(value)

    def delete(self, index):
        if index < 0 or index >= len(self.data):
            raise IndexError("Index out of bounds")
        return self.data.pop(index)

    def search(self, value):
        for i, v in enumerate(self.data):
            if v == value:
                return i
        return -1

    def traverse(self):
        for v in self.data:
            yield v

# Example usage
arr = Array()
arr.append(10); arr.append(20); arr.insert(1, 15)
idx = arr.search(20)  # 2
deleted = arr.delete(0)  # 10
print(list(arr.traverse()))

Stack + Infix/Prefix/Postfix

Stack implementation plus expression conversions and evaluations.

# Stack and expression conversion/evaluation
class Stack:
    def __init__(self): self.s = []
    def push(self, x): self.s.append(x)
    def pop(self): return self.s.pop()
    def peek(self): return self.s[-1] if self.s else None
    def empty(self): return not self.s

def precedence(op):
    return {"+":1,"-":1,"*":2,"/":2,"^":3}.get(op,0)

def infix_to_postfix(expr):
    out, st = [], Stack()
    tokens = list(expr.replace(" ", ""))
    for t in tokens:
        if t.isalnum():
            out.append(t)
        elif t == "(":
            st.push(t)
        elif t == ")":
            while not st.empty() and st.peek() != "(":
                out.append(st.pop())
            st.pop()
        else:
            while (not st.empty() and precedence(st.peek()) >= precedence(t)):
                if t == "^" and st.peek() == "^": break
                out.append(st.pop())
            st.push(t)
    while not st.empty(): out.append(st.pop())
    return "".join(out)

def infix_to_prefix(expr):
    rev = expr[::-1].replace("(", "#").replace(")", "(").replace("#", ")")
    post = infix_to_postfix(rev)
    return post[::-1]

def eval_postfix(post):
    st = Stack()
    for t in post:
        if t.isdigit(): st.push(int(t))
        else:
            b = st.pop(); a = st.pop()
            st.push({"+": a+b, "-": a-b, "*": a*b, "/": a//b, "^": a**b}[t])
    return st.pop()

def eval_prefix(pref):
    st = Stack()
    for t in pref[::-1]:
        if t.isdigit(): st.push(int(t))
        else:
            a = st.pop(); b = st.pop()
            st.push({"+": a+b, "-": a-b, "*": a*b, "/": a//b, "^": a**b}[t])
    return st.pop()

# Example
expr = "3+(4*5)-6"
post = infix_to_postfix(expr)  # "345*+6-"
pref = infix_to_prefix(expr)   # "-+3*456"
print(post, eval_postfix(post), pref, eval_prefix(pref))

Queue (FIFO)

Queue using collections.deque with enqueue, dequeue, peek.

from collections import deque

class Queue:
    def __init__(self): self.q = deque()
    def enqueue(self, x): self.q.append(x)
    def dequeue(self):
        if not self.q: raise IndexError("empty queue")
        return self.q.popleft()
    def peek(self): return self.q[0] if self.q else None
    def empty(self): return len(self.q) == 0

# Example
q = Queue()
q.enqueue(1); q.enqueue(2); q.enqueue(3)
a = q.dequeue()  # 1
b = q.peek()     # 2
print(a, b, list(q.q))

Binary Tree Traversals

Create a binary tree and traverse it using Inorder, Preorder, and Postorder.

class Node:
    def __init__(self, val, left=None, right=None):
        self.val = val; self.left = left; self.right = right

def inorder(root):
    return inorder(root.left) + [root.val] + inorder(root.right) if root else []

def preorder(root):
    return [root.val] + preorder(root.left) + preorder(root.right) if root else []

def postorder(root):
    return postorder(root.left) + postorder(root.right) + [root.val] if root else []

# Iterative versions (optional)
def inorder_iter(root):
    stack, cur, out = [], root, []
    while cur or stack:
        while cur: stack.append(cur); cur = cur.left
        cur = stack.pop(); out.append(cur.val); cur = cur.right
    return out

# Example
#       A
#      / \
#     B   C
#    / \   \
#   D   E   F
root = Node("A", Node("B", Node("D"), Node("E")), Node("C", None, Node("F")))
print("Inorder   :", inorder(root))
print("Preorder  :", preorder(root))
print("Postorder :", postorder(root))
print("InorderIter:", inorder_iter(root))
